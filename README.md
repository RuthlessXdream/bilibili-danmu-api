# Bilibili API

这个项目包含用于与Bilibili平台交互的API服务。

## 项目结构

- `bili_api/` - Bilibili API服务的主要代码
- `blivedm/` - 用于连接Bilibili直播间的客户端库

## 安装

```bash
# 克隆仓库
git clone https://github.com/RuthlessXdream/bilibili_api.git
cd bilibili_api

# 为bili_api安装依赖
cd bili_api
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
